# A. AI文案提示词模板包（首版）

## 使用说明
- 把 `{}` 内变量替换为你的信息。
- 先跑 3 版，再从中挑 1 版做人工润色。

## 模板1：小红书选题+标题

```text
你是小红书内容策划。我的账号定位是：{账号定位}；目标用户是：{目标人群}；本周目标是：{涨粉/引流/成交}。
请输出：
1) 10个选题（按“痛点-反常识-结果”结构）
2) 每个选题3个标题（口语化，避免鸡汤）
3) 每个标题给一个封面短句（10字内）
限制：不要重复，不要空话。
```

## 模板2：公众号文章骨架

```text
你是公众号编辑。主题：{主题}；读者：{读者画像}；写作目标：{教育/转化/种草}。
请输出：
- 文章标题5个
- 开头3种（故事型/数据型/问题型）
- 正文结构（3段，每段含核心观点+案例）
- 结尾CTA 2版（软引导/强引导）
```

## 模板3：短视频脚本（60秒）

```text
你是短视频编导。主题：{主题}；受众：{受众}；风格：{干货/反转/陪伴}。
请生成60秒口播脚本：
- 0-3秒钩子
- 4-35秒问题+方法
- 36-50秒案例
- 51-60秒行动引导
并附：镜头建议+字幕关键词。
```

## 模板4：评论区转化回复

```text
你是私域转化助手。用户评论：{评论内容}；我的产品：{产品名称}；价格：{价格}。
请给我3条回复：
- 1条建立信任
- 1条解决顾虑
- 1条引导私信
要求：口语化，不夸张承诺。
```

## 模板5：AI工具售卖页文案

```text
你是独立开发者的增长文案顾问。产品：{工具名称}；目标用户：{用户}；核心痛点：{痛点}；价格：{价格}。
请输出：
1) 一句话价值主张（15字内）
2) 卖点3条（功能-结果-节省）
3) 常见异议与回答4组
4) 下单按钮文案5条
```
