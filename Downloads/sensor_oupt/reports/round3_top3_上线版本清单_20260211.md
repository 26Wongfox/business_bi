## 1) 模板定价与成交优化助手（工具型）

### 核心功能（V1）
1. 输入模板类型、目标用户、竞品价格区间
2. 输出三档定价（低/中/高）+ 推荐主售档
3. 输出首发活动方案（折扣、加赠、限时）
4. 输出成交文案（标题、卖点、CTA、FAQ）

### 页面结构
- 首页：一句话价值 + 免费试用入口
- 输入页：产品信息表单
- 结果页：定价建议 + 文案建议 + 下载按钮
- 支付页：订阅/按次购买

### 定价建议
- 免费版：每日1次
- Pro版：29元/月
- 商家版：99元/月（含历史记录和复盘）

### 7天开发计划
- Day1：字段定义、输出格式定义
- Day2：定价规则v1（区间+心理价）
- Day3：Prompt链路与结果模板
- Day4：Web表单页 + 结果页
- Day5：支付与权限（免费/Pro）
- Day6：10个样本内测并修正
- Day7：上线并发布首发内容

## 2) 同城首单获客路径生成器（工具型）

### 核心功能（V1）
1. 输入城市级别、行业、可投入时间
2. 输出首单路径（7天任务清单）
3. 输出同城渠道建议（平台+动作+频率）
4. 输出私信/评论区获客话术

### 页面结构
- 首页：适合人群与使用场景
- 输入页：城市+行业+时间+预算
- 结果页：7天任务看板 + 话术包
- 复盘页：填写执行结果，自动给优化建议

### 定价建议
- 体验版：19元/次
- 月卡：49元/月
- 组合包：79元（含模板包）

### 7天开发计划
- Day1：同城渠道库初始化（抖音/快手/视频号/微信）
- Day2：行业模板（宝妈/门店/本地服务）
- Day3：路径生成逻辑与Prompt
- Day4：输入/结果页开发
- Day5：话术包接入下载
- Day6：3类人群试跑，收反馈
- Day7：发售+案例帖上线

## 3) 有流量没成交诊断器（工具型）

### 核心功能（V1）
1. 输入内容文本或商品页文案
2. 输出诊断结果（钩子、信任、CTA、价格锚点）
3. 输出改写版文案（短版/长版）
4. 输出评论区成交回复模板

### 页面结构
- 首页：问题场景（有流量没成交）
- 输入页：粘贴内容 + 目标用户
- 结果页：问题评分 + 改写建议 + 一键复制
- 升级页：人工诊断服务入口

### 定价建议
- 按次：19-39元
- 订阅：99元/月（不限次数）
- 人工诊断加购：199元/次

### 7天开发计划
- Day1：诊断维度定义（4大维度）
- Day2：评分规则+Prompt模板
- Day3：改写模板（小红书/朋友圈/商品页）
- Day4：页面与结果展示
- Day5：支付与导出（md/txt）
- Day6：内测20篇内容，优化输出质量
- Day7：上线+发布前后对比案例

## 联合上线建议（优先顺序）
1. 先上：`模板定价与成交优化助手`（变现最直接）
2. 次上：`有流量没成交诊断器`（适配大量存量内容用户）
3. 再上：`同城首单获客路径生成器`（适合3-5线/乡镇扩张）

## 联合KPI（首周）
- 访问 >= 300
- 免费试用 >= 80
- 付费 >= 20
- 首周留存 >= 30%
