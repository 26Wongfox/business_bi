# 周会打分看板（数据可视化选题永动机）

> 用法：每周一评审。先过 MECE 归类，再打分入池，再定本周发布排期。

---

## 1. 周会输入

- 候选来源：`reports/选题池_模板_2026-02-11.csv`
- 本周目标：`____条` 发布，`____条` 储备
- 主打赛道：`____`
- 实验赛道：`____`

---

## 2. 打分规则（100分）

- 新闻价值分（40）：及时性10 + 重要性10 + 急迫性8 + 重大性8 + 趣味性4
- 共鸣与变现分（45）：共鸣10 + 信息增量10 + 可行动8 + 可传播7 + 可变现10
- 执行修正分（15）：数据可得6 + 生产复杂度反向5 + 风险反向4

分层阈值：
- A池（优先生产）：`>=75`
- B池（观察储备）：`60-74`
- C池（暂缓）：`<60`

## 5. 一票否决（任一命中则退回）

- 不符合 MECE 归类（赛道或主题不清）
- 信息增量不达标（与已有内容高度重复）
- 无明确商业承接路径
- 数据来源不可追溯
- 未指定 `qa_review` 或 `publish_review` 责任人
