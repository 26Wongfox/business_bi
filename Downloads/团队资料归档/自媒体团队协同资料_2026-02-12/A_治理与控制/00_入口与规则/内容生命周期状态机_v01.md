# 内容生命周期状态机 v01

## 状态定义
1. IDEA：选题收集中
2. TODO：已立项待执行
3. DOING：创作进行中
4. REVIEW：待审核
5. SCHEDULED：已排期待发布
6. PUBLISHED：已发布
7. ANALYZING：数据回收与复盘中
8. ARCHIVED：归档完成

## 标准流转
`IDEA -> TODO -> DOING -> REVIEW -> SCHEDULED -> PUBLISHED -> ANALYZING -> ARCHIVED`

## 异常流转
- 审核驳回：`REVIEW -> DOING`
- 暂停：任意状态 -> `TODO`（需写明原因）
- 终止：任意状态 -> `ARCHIVED`（标记 Cancelled）

## 每状态必备产物
- IDEA：选题卡
- TODO：排期记录
- DOING：创作稿 v01+
- REVIEW：审核意见
- SCHEDULED：发布时间与平台
- PUBLISHED：平台回链
- ANALYZING：复盘记录
- ARCHIVED：归档索引
