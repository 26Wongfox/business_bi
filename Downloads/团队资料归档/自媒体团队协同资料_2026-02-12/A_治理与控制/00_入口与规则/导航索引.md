# ContentOps INDEX（唯一入口）

## 本周执行入口
1. 选题进入 `B_内容生产主流程/01_策划/选题池`
2. 立项后进入 `B_内容生产主流程/02_素材/资料卡`
3. 开写进入 `B_内容生产主流程/03_创作/草稿`
4. 发布前进入 `B_内容生产主流程/04_发布/排期`
5. 每周五更新 `B_内容生产主流程/05_复盘与增长/周复盘`

## 流程主线
1. `B_内容生产主流程/01_策划` -> 选题定义、优先级、评审结论
2. `B_内容生产主流程/02_素材` -> 数据来源、引用记录、素材清洗
3. `B_内容生产主流程/03_创作` -> 草稿、审核意见、定稿版本
4. `B_内容生产主流程/04_发布` -> 排期、已发布归档、平台回链
5. `B_内容生产主流程/05_复盘与增长` -> 周/月复盘与实验结果

## 横向能力
1. `C_运营对象/06_账号资产`: 账号定位与长期资产
2. `D_能力与知识资产/07_模板与自动化`: 模板和自动化规则
3. `D_能力与知识资产/08_外部研究`: 行业研究与外部资料
4. `E_归档/90_归档`: 已完成项目归档

## 单一事实源规则
- 执行中的文件只在本目录体系维护。
- `Documents/NewsVault` 仅存方法总结与知识沉淀，使用链接回指执行文件。
