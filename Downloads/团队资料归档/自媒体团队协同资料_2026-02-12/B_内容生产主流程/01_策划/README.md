# 策划（战略包）

本目录用于把选题从“灵感”变成“可持续生产系统”。

## 建议使用顺序
1. `年度议题地图与栏目矩阵_v01.md`
2. `对标媒体选题方法拆解_v01.md`
3. `选题评分模型_v01.md`
4. `选题漏斗与节奏机制_v01.md`
5. `编辑会审机制_v01.md`
6. `月度选题规划/DOING_01_ACC-ALL_月度选题规划看板_2026-02-12_v01.csv`

## 子目录说明
- `选题池`: 线索收集
- `选题评审`: 评分与立项结论
- `月度选题规划`: 当月排期与资源安排

## 执行规则
- 未经过评分模型的选题不得立项。
- 未通过会审的选题不得进入创作。
- 每周更新一次立项与排期看板。
