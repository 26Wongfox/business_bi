# 素材（数据生产能力包）

本目录用于数据发现、采集、清洗、校验、发布的全流程管理。

## 建议阅读顺序
1. `数据生产能力对标蓝图_v01.md`
2. `数据生产流水线标准流程_v01.md`
3. `智能增强数据生产手册_v01.md`
4. `数据资产版本与发布规范_v01.md`
5. `数据与引用/数据源台账_v01.csv`
6. `素材清洗/数据处理与质量校验规范_v01.md`

## 子目录
- `资料卡`: 主题背景材料
- `数据与引用`: 数据源台账与引用元数据
- `素材清洗`: 清洗规则、质检规范、处理记录

## 执行底线
- 来源不可追溯的数据不得进入创作。
- 未通过质量校验的数据不得发布。
- AI 参与环节必须保留人工复核记录。
