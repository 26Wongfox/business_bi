# 字段映射模板（选题库/内容库/复盘库）

## 1. 选题库 -> 内容库
- `选题ID` -> `选题ID`
- `账号ID` -> `账号ID`
- `账号名称` -> `账号名称`
- `选题标题` -> `内容标题(草案)`
- `负责人` -> `负责人`

## 2. 内容库 -> 复盘库
- `内容ID` -> `内容ID`
- `账号ID` -> `账号ID`
- `账号名称` -> `账号名称`
- `发布时间` -> `发布时间`
- `内容标题` -> `内容标题`

## 3. 自动化建议
- 状态=已发布 时自动创建复盘记录
- 发布时间+24h 时自动抓取指标并回填
- 复盘状态=已复盘 时自动同步到周复盘汇总表
