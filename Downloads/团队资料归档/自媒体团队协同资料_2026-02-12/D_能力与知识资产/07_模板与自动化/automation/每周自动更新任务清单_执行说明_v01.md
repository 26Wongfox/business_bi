# 每周自动更新任务清单 执行说明 v01

## 1. 执行原则
1. P0 任务优先自动化，失败必须人工补跑。
2. C级来源任务仅作为选题线索，不可直接发布。
3. 周五统一产出透明机制相关文件（引用、方法说明、QA报告）。

## 2. 触发方式
- 周一到周日固定定时任务（Cron/工作流平台）
- 每个任务失败后自动重试 2 次
- 连续失败转人工并记录原因

## 3. 与AI能力绑定
1. 摘要类：更新快讯、指标变化解释
2. 结构化类：字段映射、引用格式化
3. 质检类：异常检测、风险分级
4. 文档类：methodology 草稿、周复盘输入包

## 4. 验收指标
1. 周任务按时完成率 >= 90%
2. 失败自动恢复率 >= 70%
3. 透明机制产物覆盖率 = 100%
